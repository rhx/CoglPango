Renderer

open class Renderer : RendererProtocol

The Renderer type acts as an owner of an underlying CoglPangoRenderer instance. It provides the methods that can operate on this data type through RendererProtocol conformance. Use Renderer as a strong reference or owner of a CoglPangoRenderer instance.

  • ptr 
    Untyped pointer to the underlying `CoglPangoRenderer` instance.

    For type-safe access, use the generated, typed pointer renderer_ptr property instead.
