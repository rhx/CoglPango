RendererClassRef

public struct RendererClassRef : RendererClassProtocol

The RendererClassRef type acts as a lightweight Swift reference to an underlying CoglPangoRendererClass instance. It exposes methods that can operate on this data type through RendererClassProtocol conformance. Use RendererClassRef only as an unowned reference to an existing CoglPangoRendererClass instance.
