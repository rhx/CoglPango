RendererRef

public struct RendererRef : RendererProtocol

The RendererRef type acts as a lightweight Swift reference to an underlying CoglPangoRenderer instance. It exposes methods that can operate on this data type through RendererProtocol conformance. Use RendererRef only as an unowned reference to an existing CoglPangoRenderer instance.

  • ptr 
    Untyped pointer to the underlying `CoglPangoRenderer` instance.

    For type-safe access, use the generated, typed pointer renderer_ptr property instead.
